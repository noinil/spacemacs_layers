* Description
This layer provides the major mode for editing =CafeMol= input files.

* Install
To use this contribution add it to your =~/.spacemacs=

#+begin_src emacs-lisp
  (setq-default dotspacemacs-configuration-layers '(ct-cafemol))
#+end_src
